Money Transfer
A program for executing monetary transfers between accounts using .txt files.
Description
The project includes modules for parsing transfer files, updating account data, and generating reports.

Usage
Running the program:
Run the Main.java class in the org.example package.
Choose an operation:
1 - Parsing files: Processes all suitable files in the input directory, updates account data, and generates a report. Moves processed files to the archive directory.
2 - Listing all transfers: Displays a list of all transfers from the report file.
Input Data:
txt files in the input directory. Files must contain account numbers and transfer amounts.
Output Data:
Updated file with account data.
Report file with operations and statuses.
